Запуск проекта

Установка зависимостей

Перед запуском необходимо установить зависимости:
```
npm install
```

Запуск в режиме разработки

Команда запускает локальный сервер с горячей перезагрузкой:
```
npm run dev
```

Сборка проекта

Создание продакшен-сборки:
```
npm run build
```

Локальный предпросмотр сборки

После сборки можно запустить предпросмотр:
```
npm run preview
```
